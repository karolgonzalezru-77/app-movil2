# app-movil2
una aplicación móvil accesible, intuitiva y diseñada para mejorar la
adherencia a tratamientos médicos, que permite configurar alarmas, registrar dosis y
horarios, y enviar notificaciones a cuidadores o familiares en caso de que el paciente no
confirme la toma. La solución deberá adaptarse a usuarios con poca experiencia tecnológica,
incluir alertas claras y visibles, y ofrecer funciones de seguimiento para instituciones de salud
como EPS o ancianatos, garantizando seguridad y confidencialidad de los datos.
